### Prerequisites

- Node.js 16+

### Installation

1. **Clone the repository**

   ```bash
   git clone <your-repo-url>
   cd GreenRide
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start the app**

   ```bash
   npx expo start
   ```
